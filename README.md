# CV

Bienvenido a mi Currículum Vitae en línea.  
Soy Leandro Cruz, Analista Programador SSr. En este proyecto encontrarás mi CV profesional: experiencia, educación, habilidades técnicas, certificaciones y proyectos destacados.  
El CV es responsive, moderno y está optimizado para compartir con reclutadores o colegas.

## Estructura del proyecto

```
src/
├── components/          # Componentes React
│   ├── Header.jsx       # Encabezado con redes sociales
│   ├── InfoPersonal.jsx # Perfil y datos personales
│   ├── Experiencia.jsx  # Historial laboral
│   ├── Educacion.jsx    # Estudios y formación
│   ├── Habilidades.jsx  # Skills técnicas
│   ├── Proyectos.jsx    # Proyectos destacados
│   ├── Footer.jsx       # Pie de página
│   └── Certificados.jsx # Modal de certificados
│
├── data/                # Datos del CV (JS)
│   ├── infoPersonal.js  # Perfil y contacto
│   ├── experiencias.js  # Historial laboral
│   ├── estudios.js      # Educación formal
│   ├── formaciones.js   # Certificaciones
│   ├── habilidades.js   # Stack tecnológico
│   └── proyectos.js     # Proyectos personales
│
├── styles/             # Estilos organizados
│   ├── App.css         # Estilos principales
│   └── index.css       # Estilos globales
│
├── App.jsx             # Componente principal
└── main.jsx            # Entry point
```

## Características principales

- Interfaz moderna y responsiva
- Modo oscuro
- Animaciones suaves al hacer scroll
- Modal para visualizar certificados
- Componentes React modulares y reutilizables
- Datos separados del código (fácil actualización)
- Código simple nivel junior
